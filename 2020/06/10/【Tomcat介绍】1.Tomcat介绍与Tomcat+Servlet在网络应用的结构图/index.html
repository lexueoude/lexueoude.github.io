Tomcat是Apache 软件基金会（Apache Software Foundation）的Jakarta 项目中的一个核心项目，由Apache、Sun 和其他一些公司及个人共同开发而成.

简单而言，我们需要接受用户的访问，就需要一个服务器，Tomcat可以简单的看做是一个运行Java的服务器。

Tomcat有三个功能：

1.Web服务器

2.Jsp容器

3.Servlet容器

也就是我们可以理解成这样一个流程：

浏览器——Web服务器——Jsp、Servlet容器，*Java文件——SQL Server 、Mysql、Oracle
